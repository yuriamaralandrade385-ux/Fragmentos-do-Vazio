# Fragmentos do Vazio
RPG narrativo em texto desenvolvido em Java utilizando JOptionPane.


## Sobre

O jogo coloca o jogador dentro de uma nave espacial sem memória, onde suas decisões influenciam diretamente o rumo da história.

Cada escolha pode alterar pistas, sanidade e levar a diferentes finais.


## Funcionalidades

* Menu inicial (Jogar, Sobre, Créditos, Sair)
* Sistema de escolhas
* Múltiplos finais
* Controle de sanidade e pistas


## Tecnologias

* Java
* JOptionPane

---

## Como executar

1. Compile o código:
javac Projeto.java

2. Execute:
java Projeto
